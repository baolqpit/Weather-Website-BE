Below is the instruction to run the Weather Forecast Backend

Step 1:
```bash
git clone https://github.com/baolqpit/Weather-Website-BE.git
cd Weather-Website-BE
```
Step 2:
```bash
composer install
```
Step 3: 
```bash
cp .env.example .env
```
Step 4:
```bash
php artisan migrate
```
Step 5:
```bash
php artisan serve
```
